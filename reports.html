Metric Rules

Examples of calculated metrics.

Cost per KM = invoice_amount / actual_km
Budget Variance = actual_cost - contracted_cost
Gross Margin % = (net_sales - cost) / net_sales
Void Rate = voided_transactions / total_transactions

Sample SQL — Transportation Variance

SELECT route, carrier,
       SUM(invoice_amount) actual_cost,
       SUM(actual_km * rate_per_km) contracted_cost,
       SUM(invoice_amount) - SUM(actual_km * rate_per_km) variance_amount
FROM shipment_report_view
GROUP BY route, carrier
HAVING ABS(SUM(invoice_amount) - SUM(actual_km * rate_per_km)) > 10000
ORDER BY variance_amount DESC;

Sample SQL — POS Margin Summary

SELECT store, sales_date,
       COUNT(DISTINCT receipt_no) transactions,
       SUM(net_sales) net_sales,
       SUM(net_sales - item_cost_total) gross_margin,
       ROUND(SUM(net_sales - item_cost_total) / NULLIF(SUM(net_sales),0) * 100, 2) margin_pct
FROM pos_sales_view
GROUP BY store, sales_date
ORDER BY sales_date, store;
